1- Download the annotating tool from: https://github.com/HumanSignal/labelImg/releases/tag/v1.8.1

2- After downloading do these:
	i- 	open the predefined classes text file inside Data 
	ii- 	delete all classes, only enter your classes, e.g. "person"
	iii- 	save and close
	iv- 	Copy paste content from your other Frames folder to the current dir one.

3- run the application and click on "Open Dir"

4- navigate to your "Frames" folder

5- For YOLO click on "Pascal/VOC" icon to change to yolo annotations.

6- Use A and W key to navigate previous frames

7- Use W to create a box, drag over your label (e.g. person) 

8- A labelimg dialog box pops up, save as label name(e.g. person) and click ok

9- press ctrl+s so save as txt file.

10- repeat process for all frames
